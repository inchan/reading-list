## 요약
AI 코딩 에이전트(Claude Code, Cursor, OpenClaw 등)의 세션을 구조화된 워크플로우로 전환하는 오픈소스 상태 관리 레이어. LLM을 내장하지 않고, 워크플로우 정의·실행 추적·이벤트 기록·컬렉션 관리를 로컬 .cognetivy/ 워크스페이스에서 처리한다. npm 패키지로 설치 가능하며 Skills/MCP 호환 클라이언트와 연동된다. MIT 라이선스로 활발히 유지보수 중이다.

## 인사이트
AI 에이전트의 작업 과정을 감사 가능하고 재현 가능하게 만드는 접근이 흥미로움. 에이전트 세션이 복잡해질수록 '무엇을 했는지' 추적하는 것이 중요해지는데, 이 도구가 그 갭을 메운다. Claude Code 프로젝트에서 워크플로우 일관성을 유지하거나, 에이전트 세션을 사후 분석할 때 활용 가능하다.
