## 요약
기존 스킬, 에이전트, 커맨드를 분석하여 Claude Code의 Agent Teams(Split Pane/Swarm) 구성을 자동 생성하는 오케스트레이션 프레임워크. 27개 도메인의 137명 전문가 페르소나, Ralph Loop 품질 보장 사이클, 3계층 공유 메모리(Markdown + SQLite WAL + MCP Memory)를 포함한다. 원클릭 설치 스크립트와 슬래시 커맨드(/tofu-at)로 간편하게 실행 가능하며, Agent Office 대시보드로 실시간 진행 상황을 추적할 수 있다.

## 인사이트
Claude Code의 실험적 Agent Teams 기능을 실용적으로 활용할 수 있게 만드는 도구. 병렬 에이전트 구성 자동화와 품질 게이트(Ralph Loop, Devil's Advocate) 제공이 차별화 포인트. 대규모 코드베이스 리팩토링이나 멀티 관점 코드 리뷰에 효과적일 수 있으며, 한국어 문서가 제공되어 접근성이 좋다.
