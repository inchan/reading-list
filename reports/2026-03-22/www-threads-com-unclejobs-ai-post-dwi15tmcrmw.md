## 요약
Anthropic가 2026년 3월 20일 Claude Code Channels를 리서치 프리뷰로 공개하면서, 텔레그램·디스코드 앱에서 로컬 Claude Code 세션을 원격으로 조작할 수 있게 됐다. 이에 앞서 커뮤니티에서는 RichardAtCT의 claude-code-telegram(GitHub 2.1k stars)과 Peter Steinberger의 OpenClaw가 유사한 기능을 비공식으로 구현해 왔다. OpenClaw는 원래 Clawdbot이라는 이름이었으나 Anthropic의 상표 침해 제기로 Moltbot을 거쳐 OpenClaw로 두 번 개명됐으며, 이 과정이 업계에서 화제가 됐다. 공식 Channels 플러그인은 MCP 서버 방식으로 동작하며, 현재 프리뷰 단계라 Anthropic 승인 플러그인만 허용된다. 정식 출시 시 더 많은 플랫폼 지원과 커뮤니티 채널 확장이 예고되어 있다.

## 인사이트
Anthropic가 상표 분쟁으로 커뮤니티 프로젝트를 억제한 뒤 직접 동일 기능을 공식화한 것은 의미심장하다. 개발자는 터미널 앞에 상주하지 않아도 모바일에서 Claude Code 세션을 비동기로 제어할 수 있게 되어, AI 에이전트를 '항시 실행(always-on)' 방식으로 운영하는 워크플로우가 현실화된다. 현재 프리뷰 단계이므로 Pro/Max/Enterprise 계정과 Claude Code v2.1.80 이상이 필요하며, API 키 인증은 지원하지 않는다는 점을 유의해야 한다.
