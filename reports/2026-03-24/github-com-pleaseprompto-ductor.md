## 요약
ductor는 PleasePrompto가 개발한 Python 도구로, Telegram 또는 Matrix 메신저를 통해 Claude Code·Codex CLI·Gemini CLI를 원격으로 제어할 수 있게 해준다. 실시간 응답 스트리밍, 지속적 메모리, 크론 잡 스케줄러, 웹훅, Docker 샌드박싱 등 고급 자동화 기능을 지원하며 PyPI를 통해 설치 가능하다. 듀얼 허용 목록(allowed_user_ids + allowed_group_ids) 보안 모델로 외부 접근을 안전하게 통제하며, Telegram 포럼 모드나 Matrix 룸마다 독립적인 CLI 컨텍스트를 유지한다. 여러 CLI 도구에 걸쳐 스킬을 공유하는 크로스 툴 스킬 동기화 기능도 제공한다. 출시 후 약 한 달 만에 32개 릴리스를 기록하며 활발히 개발 중이다.

## 인사이트
스마트폰에서 AI 코딩 작업을 모니터링·승인·제어하는 새로운 워크플로우를 가능하게 하며, 장시간 실행 배치 작업이나 자동화 파이프라인 운영에 특히 유용하다. ccgram, untether 등 유사 프로젝트와의 경쟁 속에서 크론 잡과 Docker 샌드박싱까지 통합한 것이 차별점이다.
