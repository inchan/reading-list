## 요약
Ruflo(구 Claude Flow)는 ruvnet이 개발한 엔터프라이즈급 AI 에이전트 오케스트레이션 플랫폼으로, Claude Code를 멀티 에이전트 개발 환경으로 확장한다. 259개의 MCP 도구와 60개 이상의 전문 에이전트 스웜을 지원하며, 10개월·5,800개 이상의 커밋을 거쳐 2026년 2월 27일 v3.5.0 첫 안정 배포판을 출시했다. TypeScript와 Rust/WASM으로 구축된 정책 엔진과 최대 75% API 비용 절감이 가능한 3단계 지능형 모델 라우팅을 제공한다. HNSW 벡터 메모리 기반 자기학습(SONA), 암호학적 증명 기반 쓰기 보호(MutationGuard), 드리프트 방지 기능 등이 포함됐다. npm으로 `npx ruflo@latest init --wizard` 명령으로 빠르게 시작할 수 있으며 기존 `claude-flow` 별칭도 지속 지원된다.

## 인사이트
단순 코파일럿을 넘어 실제 엔지니어링 팀처럼 AI 에이전트 스웜을 조직화·자기학습시키는 새로운 개발 패러다임을 제시한다. API 비용 75% 절감을 내세우는 3단계 모델 라우팅과 WASM 기반 경량 실행 엔진은 대규모 에이전트 운용 시 실질적인 비용 최적화 수단이 될 수 있다.
