## 요약
TradingAgents는 UCLA·MIT 연구자들이 개발한 멀티 에이전트 LLM 금융 거래 프레임워크로, 실제 헤지펀드 조직 구조를 AI 에이전트로 구현한다. 기본·감성·뉴스·기술 분석가 4팀이 병렬로 시장 정보를 수집하고, 리서처가 종합한 뒤 트레이더가 결정하면 리스크 매니저 검토 후 펀드 매니저가 최종 승인하는 5단계 워크플로우다. GitHub에서 39,300개 이상의 스타와 7,300개 이상의 포크를 기록하며 오픈소스 금융 AI 분야의 대표 프로젝트로 자리잡았다. LangGraph 기반으로 OpenAI·Google·Anthropic·xAI 등 다양한 LLM 공급자를 지원하며, v0.2.0(2026년 2월)부터 Claude 4.x도 공식 지원한다. GPU 없이 API 크레딧만으로 실행 가능하며 arXiv 논문(2412.20138)으로 학술 검증되었다.

## 인사이트
금융 도메인에서 검증된 역할 분담 멀티 에이전트 아키텍처는 다른 복잡한 의사결정 도메인(법률 분석, 의료 진단 등)에도 참고할 수 있는 설계 패턴을 제공한다. Claude API 공식 지원으로 Anthropic 생태계 내에서 즉시 실험 가능하며, 활발한 커뮤니티(7,300+ 포크)를 기반으로 파생 프로젝트와 추가 전략 모듈이 빠르게 개발되고 있다.
