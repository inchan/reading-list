## 요약
Grok CLI는 xAI의 Grok 모델을 터미널에서 사용할 수 있는 오픈소스(MIT) 코딩 에이전트로, npm i -g grok-dev 한 줄로 설치된다. Grok의 실시간 X·웹 검색, 멀티 에이전트 위임, Telegram 원격 제어, 1M 토큰 컨텍스트 윈도우, 이미지·영상 생성 등을 지원한다. 다만 이 프로젝트는 xAI 공식 제품이 아니라 슈퍼에이전트(superagent-ai)가 만든 커뮤니티 빌드 비공식 도구임을 주의해야 한다. Claude Code 대비 '실시간 X 데이터 접근'을 차별점으로 내세우며 Claude Code Alternatives로도 소개되고 있다. API 비용은 grok-code-fast-1 기준 입력 $0.20/1M 토큰으로 비교적 저렴하다.

## 인사이트
xAI 공식 CLI가 아닌 커뮤니티 프로젝트임에도 2.5K+ 스타를 기록하며 빠르게 성장 중인 점은, Claude Code 생태계 외부에서 유사한 agentic coding 도구가 활발히 등장하고 있음을 시사한다. Telegram 원격 제어와 실시간 X 검색은 Claude Code에 없는 차별화 기능으로, 모바일 원격 작업 시나리오에 관심 있는 개발자에게 실용적 대안이 될 수 있다.
