# Architecture Diagram Skill in Claude Code (Hermes)

Claude Code (internally referred to as "Hermes") supports installable skills that extend its slash-command vocabulary. A skill for generating architecture diagrams has been noted, accessible via the `/architecture-diagram` prompt once the skill is installed.

## How It Works

1. **Install the skill** — obtain the architecture-diagram skill inside Hermes.
2. **Invoke the command** — use `/architecture-diagram` as a slash prompt to have Claude generate a diagram of your system or codebase architecture.

The feature was surfaced in a Korean-language Threads post ([@ai.corder](https://www.threads.com/@ai.corder/post/DXIzTBvmUmX)) noting: *"Hermes에서는 스킬 받아서 /architecture-digram 프롬프트로도 가능하다고 하네요."* ("It seems you can also do it in Hermes by receiving the skill and using the /architecture-diagram prompt.")
