# 문화와 참고자료 대기열

이 페이지는 아직 독립 문서로 만들 만큼 내용이 충분하지 않은 문화/참고자료 북마크를 보존한다. 본문이 부족한 항목은 추정으로 채우지 않고 원문 확인 대상으로 남긴다.

## 영상과 드라마

영국 넷플릭스 드라마 소스는 이미지 기반 게시물에서 "영국의 한적한 마을", "17살 여학생 살해", "5년 뒤 진실 추적", "소설 원작", "시청 뷰수 19,500,000" 같은 일부 텍스트만 추출되어 있다. 작품명과 전체 목록은 원문 확인이 필요하다. [raw](../raw/raindrop/items/857570762/20260322T2251525.8a1ef12ca5976ccfb0618857adcea33e6161b8d9c9a5609de00b613fdbceb3a2.md)

여름캠프 도플갱어 게시물은 "할리", "애니", 여름캠프 장면이 언급된 영화/짤 소스다. 제목과 맥락을 확정하려면 원문 확인이 필요하다. [raw](../raw/raindrop/items/857569407/20260322T2251525.3fa483afad4ef5fa72081bbb817b1736afeb1b99c5d300ee929fe8552bcac49f.md)

## 검색과 북마크 잔여물

유발 하라리 네이버 통합검색 소스는 검색 결과 캡처만 남아 있다. 특정 책, 강연, 인터뷰 중 무엇을 저장하려 했는지 불명확하므로 원문 확인 또는 삭제 후보 검토가 필요하다. [raw](../raw/raindrop/items/379447627/20260322T1538139.3d4c6928a12ce82c21cfdab2423da60eeccfefeb35214fce5f26ccd16bfd4797.md)

Safari 책갈피 HTML 소스는 제목과 파일명만 남고 본문이 추출되지 않았다. 전체 북마크 덤프인지, 개별 저장 항목인지 확인이 필요하다. [raw](../raw/raindrop/items/316446573/20260322T2251512.48cf1892bd0fb671f1891a4e890257d3aa7e015e36f9fc101cd1fd886c7fb7f9.md)
