AI 숏츠 영상 자동 생성기
요청 많으셨던 숏츠 자동화 소스 공개합니다.
명령어 하나로 유튜브 숏츠 / 릴스 / 틱톡 영상이 뚝딱.

주제 입력하면 끝.
→ Gemini가 콘텐츠 자동 생성
→ TTS 나레이션 자동 생성 (무료)
→ 배경 + 자막 슬라이드 자동 디자인
→ Ken Burns 효과로 영상 합성까지 자동
비용? 전부 무료입니다.
Gemini 무료 티어 + edge-tts(무료) + ffmpeg(무료)
단순한 구조로 만들어놨으니 입맛대로 수정해보시는 걸 추천드립니다.
테마 색상, TTS 음성, 읽기 속도 등 config 파일 하나로 커스터마이징 가능합니다.

추후에 웹 UI로 딸깍 한 번이면 영상 나오도록 업데이트 예정입니다.
Windows / Mac / Linux 전부 지원하고,
설치 가이드 꼼꼼하게 적어놨으니 시도해보시고 어려우면 디엠주세요! 😎
