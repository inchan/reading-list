## Excerpt

요즘 가장 핫한 AI 오픈소스 "ClawRouter"를 아시나요?

AI 비서 쓰다가 날아온 청구서 폭탄에 당황했던 분들을 위한 'AI 비용 절감 시스템'입니다.

0.1초의 판단으로 내 지갑을 지키는 핵심 기능을 요약해 드릴게요.

ClawRouter는 BlockRunAI가 만든 오픈소스 LLM 라우터입니다.

앱과 30개+ LLM 제공자(OpenAI, Anthropic, Google, DeepSeek, xAI 등) 사이에 앉아서, 각 요청을 분석하고 “제일 싼데도 감당 가능한 모델”로 자동으로 라우팅해줍니다.

## Note

ClawRouter는 BlockRunAI가 만든 MIT 라이선스 오픈소스 LLM 라우터로, 앱과 55개 이상의 LLM 제공자 사이에 로컬 프록시(포트 8402)로 동작한다.
각 요청을 15개 차원의 가중치 스코어로 평가하여 1ms 이내에 가장 저렴하면서도 충분한 성능의 모델로 라우팅하며, API 비용을 최대 92%까지 절감할 수 있다고 주장한다.
기존 LLM 라우터와 달리 에이전트 네이티브 설계를 채택하여 API 키 대신 지갑 서명 인증과 x402 프로토콜 기반 USDC 마이크로 결제를 사용한다.
OpenClaw(Claude Code 오픈소스 변형), ElizaOS 등 OpenAI 호환 API를 사용하는 모든 에이전트와 통합 가능하며, 속도 제한·과금 오류·서비스 장애 발생 시 최대 3단계 자동 폴백 체인도 지원한다.

메모: AI API 비용 폭증 문제를 인프라 레이어에서 해결하는 접근이 흥미롭다.
