## Note

paoloanzn/free-code는 2026년 3월 31일 Anthropic npm 패키지에서 실수로 유출된 Claude Code 소스 코드를 기반으로 한 오픈소스 포크를 소개하는 Threads 게시물이다.

관련 스레드에서는 유출 소스코드에서 발견된 44개 피처 플래그와 ULTRAPLAN 기능도 소개한다.
공개 직후 2.4k 스타를 빠르게 기록했다.
