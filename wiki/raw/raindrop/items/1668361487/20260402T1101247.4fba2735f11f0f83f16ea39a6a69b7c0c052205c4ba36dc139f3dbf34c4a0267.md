## Note

last30days-skill은 어떤 주제든 Reddit, X(트위터), YouTube, Hacker News, Polymarket, Bluesky, TikTok, Instagram 등 최대 10개 소스에서 최근 30일간의 정보를 수집·분석·요약해주는 Claude Code 스킬이다.
v2.9.5 기준 17.4k GitHub 스타를 보유한 인기 오픈소스 프로젝트로, `/plugin marketplace add mvanhorn/last30days-skill` 한 줄로 설치할 수 있다.
Polymarket 예측 시장 데이터와 HN 커뮤니티 반응을 결합한 다중 신호 품질 랭킹이 특징이며, X 핸들 자동 해석, 스마트 서브레딧 발견, 주제 비교 모드(comparative mode) 등 고급 기능을 제공한다.
API 키 없이도 Reddit·HN·Polymarket 3개 소스를 즉시 사용 가능하며, Gemini CLI 및 OpenAI Codex CLI와도 호환된다.

인사이트
단순 검색 도구를 넘어 '커뮤니티가 실제로 관심을 갖고 베팅하는 것'을 파악하는 실시간 인텔리전스 도구다.
