## Excerpt

Implement OpenSpec tasks effortlessly with the OpenSpec Apply Change skill for Claude Code. Automate task execution, track progress, and stay in spec.

## Note

OpenSpec Apply Change는 Claude Code에서 OpenSpec 스펙 주도 개발 워크플로우의 구현(Apply) 단계를 자동화하는 스킬이다.
활성 변경 사항을 자동으로 탐지·선택하고, 스키마별 지침을 파싱하여 코드 업데이트를 실행하면서 설계 문서와의 정렬을 유지한다.
구현·검증·진행 보고의 루프를 자동화하여 단계 간 컨텍스트 손실 없이 계획대로 기능을 구현할 수 있도록 지원한다.
차단 요소나 설계 불일치 발생 시 내장 가드레일이 처리한다.

인사이트
OpenSpec의 수동 Apply 단계를 자동화함으로써 스펙 주도 개발의 실무 적용 마찰을 크게 줄일 수 있다.
