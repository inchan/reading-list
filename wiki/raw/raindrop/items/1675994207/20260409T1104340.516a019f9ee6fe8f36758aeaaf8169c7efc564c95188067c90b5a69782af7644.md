## Note

qmd는 마크다운 문서, 회의록, 지식베이스 등을 로컬에서 인덱싱하고 검색하는 CLI 도구로, BM25 전문 검색·벡터 의미 검색·LLM 재랭킹을 결합한 하이브리드 파이프라인을 완전 로컬에서 실행한다.
파인튜닝된 1.7B 쿼리 확장 모델로 쿼리 변형을 생성해 병렬 검색 후 RRF(Reciprocal Rank Fusion)로 합산하고 Qwen3-Reranker로 최종 재랭킹한다.
node-llama-cpp를 통해 세 종류의 GGUF 모델(임베딩·재랭킹·쿼리 확장)을 HuggingFace에서 자동 다운로드해 로컬 실행하며, AST 기반 코드 청킹, MCP 서버, Claude Code 플러그인, HTTP transport 데몬 모드 등 에이전트 워크플로우 통합도 잘 갖춰져 있다.
SDK로도 제공되어 Node.js/Bun 애플리케이션에 직접 임베드할 수 있다.

인사이트
쿼리 확장 모델을 직접 파인튜닝한 점이 일반 RAG 도구와의 핵심 차별점이다.
