## Note

Understand-Anything은 어떤 코드베이스나 지식 베이스(Karpathy LLM wiki 등)를 탐색 가능한 인터랙티브 지식 그래프로 변환하는 멀티플랫폼 AI 플러그인이다.
5개 전문 에이전트가 병렬로 파일·함수·클래스·의존성을 추출하고 구조 그래프와 비즈니스 도메인 그래프 두 가지 뷰를 제공한다.
Claude Code, Codex, Cursor, GitHub Copilot, Gemini CLI, OpenCode, OpenClaw, Antigravity, Pi Agent 등 10개 이상의 플랫폼을 지원하며, 각 플랫폼별 설치 방법이 정비되어 있다.
퍼지·시맨틱 검색, 가이드 투어, diff 영향 분석, 페르소나 적응형 UI 등의 부가 기능도 내장되어 있다.

인사이트
대규모 코드베이스(20만 줄 이상)에 새로 합류한 개발자의 온보딩 시간을 크게 단축할 수 있는 실용적인 도구다.
