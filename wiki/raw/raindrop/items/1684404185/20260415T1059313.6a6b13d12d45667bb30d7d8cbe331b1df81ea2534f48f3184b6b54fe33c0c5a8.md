## Excerpt

요즘 간간히 Codex나 Claude Code 혹사시키시는 분들 계십니다. 
RAM이나 CPU 터지는 스샷과 함께요.

근데 사실 대부분의 경우 AI 도구 자체가 무거운 게 아닙니다.

 세션 끝나고 남은 고아 프로세스가 범인이에요.

 MCP 서버, 서브에이전트, 헤드리스 브라우저, esbuild 같은 자식 프로세스들이
 세션 종료 후에도 안 죽고 쌓입니다. Activity Monitor 열어보시면 node
 프로세스가 수십 개 떠있을 겁니다.

 이걸 자동으로 정리하는 CLI를 소개합니다.  좀비클린 -zclean

 npx @thestackai/zclean init

 한 줄이면 끝입니다.

 - 세션 종료 시 자동 정리 (SessionEnd hook)
 - 매 시간 백그라운드 스캔 (혹시 놓친 것 정리)
 - 실행 중인 세션은 절대 안 건드림
 - macOS / Linux / Windows
 - 의존성 0개
